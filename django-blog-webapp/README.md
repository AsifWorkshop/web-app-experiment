# Django Blog Web App

A full-stack blogging platform built with Django. This project is part of my **web-app-experiment** mega-repo.

## 🚀 Features
- **CRUD Functionality:** Create, Read, Update, and Delete blog posts.
- **User Authentication:** Secure login and registration system.
- **Messaging System:** (Planned) Real-time interaction between users.
- **Responsive Design:** Clean UI for both mobile and desktop.

## 🛠️ Tech Stack
- **Backend:** Python, Django
- **Database:** SQLite (Development)
- **Environment:** Python Virtual Environment (`venv`)

## ⚙️ How to Run Locally
1. Navigate to this directory: `cd django-blog-webapp`
2. Activate venv: `.\venv\Scripts\activate`
3. Install dependencies: `pip install django`
4. Run server: `python manage.py runserver`
